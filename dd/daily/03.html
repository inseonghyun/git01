  • 글씨 기울림
    • em : 강조하는 의미
    • i : 그냥 기울림

  • 볼드체
    • strong : 강조하는 의미
    • b : 그냥 bold체

  • alt의 3가지 속성
    • 설명을 적는(접근성 향상)
    • 검색엔진 반영seo
    • 사진이 보이지 않을 때 alt속성이 뜸
    • 블럭 : 공간을 갖는다
    • 인라인 : 이미지 크기에 맞게 설정(공간X)

  • mark : 마킹

  • small : 작은글씨체

  • sub : 글씨를 위

  • sub : 글씨를 아래

  • 태그 안에 title : 마우스를 올려보세요(모든 태그에 중복사용가능)

  • 목록을 표시하는 태그
    • ol : 그룹(순서있음)
    • ui : 불릿총알(순서없음)
    • li : 리스트(단독사용불가)개별항목
    • dl : 설명목록 : dl>dt+dd(세트, dd는 여러개 사용가능)
